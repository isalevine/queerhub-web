# QueerHub
QueerHub is a blogging platform centering queer and trans content!

# Setup
1. Clone repo.
1. In `queerhub-web` directory, run `rails s` to start Rails server.

# Features
## Event Sourcing
* Changes to state and data are recorded in immutable event objects
